# Fenny — Project Memory: agentic-memories

## Project Overview
Agentic memory system with typed structured memory (episodic, emotional, portfolio), vector search via ChromaDB, and a user profile layer. FastAPI backend, React/TS/Vite UI, Postgres/TimescaleDB, Redis. Docker Compose. Primary user: Ankit (Telegram user 1169491032).

## Stack
- Backend: FastAPI, Python, ChromaDB (vectors), TimescaleDB (typed tables), Redis (cache)
- Frontend: React + TypeScript + Vite + TanStack Query
- Ingestion: LangGraph-style unified graph in `src/services/unified_ingestion_graph.py`
- Ports: API 8080, UI 3000, TSDB 5432, ChromaDB 8000, Redis 6379

## Build/Test Commands
- `make start` — bring up full stack
- `make migrate` — run DB migrations (check `PG_PORT=5432` if it fails)
- `curl http://localhost:8080/health` — API liveness
